# **캐나다 위치기반 커뮤니티 & 구인구직 플랫폼**

> **“Your neighborhood, your opportunities.”**
> 
> 
> 캐나다 한인·유학생·워홀러·소상공인을 위한 **동네 기반 구인/커뮤니티 플랫폼**
> 
- 프론트엔드/백엔드: Next.js
- 배포: Vercel 무료 배포 가능
- DB: superbase

---

## 1️⃣ 문제 정의 (Problem)

캐나다 현지 한인/워홀러/유학생은:

- **동네 기반 구인 정보 부족** → 캐스모, Indeed, linkedIn, Facebook Groups UX/신뢰 문제
- **커뮤니티 정보 분산** → 카톡, 다음카페, FB 그룹 등
- **단기 일자리 연결 어려움** → 기존 Job Board 위치 기반 필터 약함
- **거래 안전성 부족** → 공동구매/중고거래 시 사기 우려

➡ **결론:** “동네 기반으로 신뢰할 수 있는 구인/커뮤니티/거래 플랫폼이 없다.”

---

## 2️⃣ 해결책 (Solution)

**위치 기반 + 구인/구직 + 커뮤니티 + 채팅 + 프로필**을 통합한 앱입니다.

- 내 위치 반경 3~10km 내 구인/구직 정보 노출
- 신뢰 기반 사용자 리뷰 및 프로필
- 카테고리별 커뮤니티 & 공동구매/중고거래
- 실시간 1:1 및 그룹 채팅

---

## 3️⃣ IA (Information Architecture)

```
App
│
├─ Onboarding / 로그인
│   ├─ 이메일/전화번호 로그인
│   ├─ 소셜 로그인 (Apple / Google / Kakao)
│   └─ 위치 권한 요청
│
├─ 홈 (Feed)
│   ├─ 내 위치 기반 게시글 (리스트뷰/지도뷰)
│   │   ├─ 구인/구직
│   │   ├─ 커뮤니티
│   │   └─ 공동구매/중고거래
│   ├─ 필터 (카테고리, 거리, 최신/인기)
│   └─ 검색 (키워드, 사용자, 게시글)
│
├─ 구인/구직
│   ├─ 카테고리별 탐색
│   ├─ 게시글 작성
│   ├─ 지원하기 → 채팅 연결
│   └─ 내 지원 내역
│
├─ 커뮤니티
│   ├─ 카테고리별 (Housing / Events / Tips / Buy & Sell)
│   ├─ 게시글 작성
│   └─ 댓글 / 그룹 채팅
│
├─ 채팅
│   ├─ 1:1 채팅
│   ├─ 그룹 채팅
│   └─ 사진/파일/위치 공유
│
├─ 프로필
│   ├─ 사용자 정보 (사진, 이름, 위치, 자기소개)
│   ├─ 활동 내역 (게시글, 댓글, 지원 기록)
│   ├─ 리뷰 & 평점
│   └─ 설정 (알림, 위치, 계정 관리)
│
├─ 공동구매/중고거래
│   ├─ 위치 기반 게시글
│   ├─ 가격/카테고리 필터
│   └─ 채팅 연결
│
└─ 알림
    ├─ 구인/구직 지원
    ├─ 댓글 & 좋아요
    └─ 시스템 알림
```

## 4️⃣ 기능 상세

| 기능 | 설명 |
| --- | --- |
| 위치 기반 피드 | GPS 기준 반경 3~10km 게시글 노출 |
| 구인/구직 | 카테고리별 게시글 작성, 지원, 채팅 연결 |
| 커뮤니티 | 카테고리별 게시글 + 댓글, 그룹 채팅 |
| 채팅 | 1:1 + 그룹, 사진/파일/위치 전송 |
| 프로필 | 사진, 자기소개, 리뷰, 평점, 활동 내역 |
| 공동구매/중고거래 | 위치 기반, 카테고리/가격 필터, 채팅 연결 |
